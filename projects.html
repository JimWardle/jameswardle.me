Leaf Spy Analyzer

A fast-turnaround SaaS for Nissan Leaf owners. Users upload a Leaf Spy video and receive an AI-generated battery health report, including predicted SOH, warranty risk, degradation stats, and resale impact. Built in under 20 hours using ffmpeg, OpenAI API, and Flask, with Stripe integration and CI/CD to production.

  • Video OCR → text → structured prompt → GPT-4o → PDF report
  • Live in production with first sales within 2 weeks
  • 99.5% profit margin at £3.99 per report

CloudFlow Monitoring & Dashboard

A full enterprise solution for monitoring 125+ Microsoft Power Platform environments, visualised in a live React dashboard. Built for Mercury, this system tracks 2,000+ flows across tenants, filters noise intelligently, and syncs data via a fully automated PowerShell-to-Git pipeline.

  • PowerShell pipeline scanning 17,000+ flows and syncing via Git to production
  • Branded React dashboard with real-time filtering and CSV/XLSX export
  • Azure Static Web Apps deployment with CI/CD and global CDN
  • Zero manual steps — updates run on a schedule and push live automatically
