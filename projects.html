Project Showcase

SuperCrumble – Crypto Portfolio Analytics

Built in 48 hours as a performance challenge, SuperCrumble reduced crypto portfolio analysis times from minutes to under a second. Powered by PHP, MySQL, and GitHub Actions, it was designed for speed, accuracy, and ease of deployment.

This was a solo sprint from concept to delivery, with a focus on minimal latency, clean UX, and automating routine data handling tasks.

DiceDecider.com – Game Night Helper

A fast, lightweight tool for indecisive board game groups. Pulls live data from the BoardGameGeek API to recommend games based on player count, play time, and mood. Built for fun, but coded cleanly and deployed with care.

Peg It – Weather-Based Laundry Advisor

A simple tool built in one afternoon to help decide whether it’s worth hanging washing out. It interprets weather data into plain advice — sunny, safe, or sack it off. A fun side project grounded in real-life usefulness.
